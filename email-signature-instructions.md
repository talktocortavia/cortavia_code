# CORTAVIA Email Signature Setup Instructions

## Email Signature Files

1. **email-signature.html** - HTML version for modern email clients (Gmail, Outlook, Apple Mail, etc.)
2. **email-signature-plain.txt** - Plain text version for basic email clients

## How to Add Signature to Your Email Client

### Gmail
1. Open Gmail → Settings (gear icon) → See all settings
2. Go to "General" tab
3. Scroll down to "Signature"
4. Click "Create new"
5. Copy the HTML code from `email-signature.html` (the table section)
6. Paste it into the signature editor
7. Click "Save changes"

### Outlook (Desktop)
1. File → Options → Mail
2. Click "Signatures..."
3. Click "New" to create a signature
4. Name it "CORTAVIA"
5. Copy the HTML code from `email-signature.html`
6. Paste it into the signature editor
7. Click "OK"

### Outlook (Web)
1. Go to Settings (gear icon) → View all Outlook settings
2. Mail → Compose and reply
3. Under "Email signature", paste the HTML code
4. Click "Save"

### Apple Mail (Mac)
1. Mail → Preferences → Signatures
2. Click "+" to create new signature
3. Name it "CORTAVIA"
4. Copy the HTML code from `email-signature.html`
5. Paste it into the signature editor
6. Close preferences

### Thunderbird
1. Tools → Account Settings
2. Select your email account
3. Click "Manage Identities"
4. Edit your identity
5. Go to "Attach signature" section
6. Copy HTML code from `email-signature.html`
7. Paste and save
