# Structure

## Directories

- `frontend/` - React source (development)
- `backend/` - PHP API (optional)
- `public/` - Build output (deploy this)

## Key Points

- `public/` = Frontend build only
- `backend/` = API (separate)
- `public/index.php` = Serves React app (not the API)

## Deploy

Upload `public/` to web server.
