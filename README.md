# CORTAVIA Website

React + PHP application.

## Quick Start

```bash
# Backend
cd backend && php -S localhost:8000 index.php

# Frontend
cd frontend && npm install && npm run dev
```

## Build

```bash
cd frontend && npm run build
```

**Note**: Built files are committed to Git for easy deployment.
